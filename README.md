# HELP-NGO-COUNTRY-CLUSTERING
App Link : https://ngo-country-clustering-app.streamlit.app/

Country Development Status Classification using Unsupervised Learning

This project uses unsupervised machine learning to classify countries as underdeveloped, developing, or developed based on socio-economic and health indicators. The analysis supports HELP International, an NGO aiming to allocate $10 million in aid to the countries most in need.
